RISC-V Machine

Designed and implemented a pipelined RISC-V processor using SystemVerilog on an Intel FPGA with an integrated ARM Cortex host. Engineered a 5-stage pipeline ARM architecture to optimize execution speed and reduce clock cycle stalls by over 30%.

Coin-Collecting Robot

Designed and built a dual-mode (manual and automatic) robot capable of detecting, collecting, and storing metallic coins using inductive sensing and an electromagnet arm. Achieved over 90% detection accuracy and reduced average collection time to under 3 seconds per coin.

SUBC Electronics System

Electrical Engineer for UBC's Human Powered Submarine team. Programmed BNO055 sensor via I2C to stream Euler angle and accelerometer data at 100 Hz, enabling closed-loop motion adjustments that improved submarine stability and directional control by an estimated 30%.
